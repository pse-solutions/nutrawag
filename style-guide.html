Responsive Design

The NutraWag website is designed to be fully responsive, with breakpoints at the following widths:

  • Mobile: Up to 768px
  • Tablet: 769px to 1024px
  • Desktop: 1025px and above

Key responsive changes include:

  • Feature icons stack in a 2x2 grid on mobile
  • Product container switches to a column layout on mobile
  • Benefits list changes from 2 columns to 1 column on mobile
  • Font sizes and spacing are reduced proportionally on smaller screens
